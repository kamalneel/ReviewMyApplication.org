## Research Framework for Each College

For each institution, capture:
1. **Source URLs**
   - Official admissions blog
   - Admissions officer statements
   - Application guides
   - Essay prompts and guidance
   - Admitted student profiles

2. **Counselor/Reviewer Perspectives**
   - What they value in applicants
   - How they evaluate applications
   - Holistic review factors
   - Academic vs. extracurricular weight

3. **Application Components Analysis**
   a. **Academic Criteria**
      - GPA expectations
      - Test score ranges/requirements
      - Course rigor importance
      - Academic interests
   
   b. **Essays**
      - What makes strong essays
      - Common pitfalls
      - Authenticity markers
      - Topic preferences
   
   c. **Extracurriculars**
      - Depth vs. breadth
      - Leadership expectations
      - Impact demonstration
      - Unique activities valued
   
   d. **Letters of Recommendation**
      - What reviewers look for
      - Ideal recommender profiles
      - Content that matters
   
   e. **Demonstrated Interest**
      - Importance level
      - How it's measured
      - Impact on admissions

4. **Evaluation Rubrics**
   - How applications are scored/rated
   - Decision-making frameworks
   - Committee processes

## Output Format
JSON structure with:
- University metadata
- Source URLs with descriptions
- Categorized insights by application component
- Direct quotes from admissions officers (with sources)
- Evaluation frameworks
